# CityPostBot 🤖

Помощник по постингу La Ciudad de los Sentidos в Telegram/Facebook группах.

## Что умеет

- Хранит список групп (название, платформа, формат, ссылка)
- Ротирует версии объявлений A/B/C автоматически
- Записывает каждую публикацию с датой
- Отслеживает отклики
- Помечает мёртвые группы (3 публикации без отклика)
- Показывает статистику по версиям

## Команды

| Команда | Что делает |
|---------|-----------|
| /start | Приветствие и список команд |
| /groups | Список всех групп со статусом |
| /add | Добавить новую группу |
| /post | Получить текст для публикации |
| /stats | Статистика по всем группам и версиям |
| /setlink | Изменить ссылку в объявлениях |
| /setdate | Изменить дату игры |

## Формат добавления группы

```
название | @handle | Telegram или Facebook | текст или картинка
```

Пример:
```
Испания чат СНГ | @spainchats | Telegram | текст
```

## Запуск

### Локально
```bash
pip install -r requirements.txt
export BOT_TOKEN="твой_токен"
export ADMIN_ID="твой_telegram_id"
python bot.py
```

### На сервере (Railway / Render)
Переменные окружения:
- `BOT_TOKEN` — токен от BotFather
- `ADMIN_ID` — твой Telegram ID (узнать: @userinfobot)

## Как узнать свой ADMIN_ID

Напиши @userinfobot в Telegram — он пришлёт твой числовой ID.
